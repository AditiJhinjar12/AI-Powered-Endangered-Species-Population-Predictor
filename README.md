# 🦁 AI-Powered Endangered Species Population Predictor

An AI-driven web application that predicts future population trends of endangered species using historical population records and environmental factors. The system leverages machine learning to forecast population changes, assess extinction risk, and provide conservation recommendations to support researchers, wildlife organizations, and conservation authorities.

## 🚀 Features

* Predict future population of endangered species using machine learning.
* Analyze historical population and environmental data.
* Classify extinction risk (Safe, Vulnerable, Endangered, Critically Endangered, Near Extinction).
* Interactive dashboard with charts and analytics.
* AI-based conservation recommendations.
* Species and dataset management through an admin panel.
* Upload and process CSV datasets.
* Generate downloadable prediction reports.
* Secure user authentication and role-based access.
* Responsive web interface for desktop and mobile devices.

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Bootstrap 5, JavaScript
* **Backend:** Python, Flask
* **Machine Learning:** Pandas, NumPy, Scikit-learn (Random Forest, XGBoost, Linear Regression), TensorFlow/Keras (optional)
* **Database:** SQLite / MySQL
* **Visualization:** Chart.js
* **Version Control:** Git & GitHub

## 📊 Project Workflow

1. Upload historical and environmental datasets.
2. Preprocess and clean the data.
3. Train the machine learning model.
4. Predict future species population.
5. Classify extinction risk.
6. Generate AI-powered conservation recommendations.
7. Visualize results through an interactive dashboard and reports.

## 🎯 Objective

The goal of this project is to support wildlife conservation by providing accurate population predictions and actionable insights, enabling informed decisions for protecting endangered species.
